It is a project that build for people who want to search song lyrics

## Built with
- Reactjs
- Nodejs

## Project Features
- Login
- Register
- Search
- Get lyric
- Favorite

## Getting Started

1.Clone the repo
```
https://github.com/Pongsathon346/M8_Project.git
```
2.Run application
Frontend
```
$ cd react-app
$ npm install #
$ npm start
```
Backend
```
$ cd-server
$ npm install #
$ node index
```
Replace # with NPM Packages below
